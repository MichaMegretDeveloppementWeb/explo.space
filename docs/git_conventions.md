# Conventions Git — Conventional Commits

## Préfixes
- `feat:` fonctionnalité ; `fix:` correction ; `refactor:` interne ; `test:` ; `docs:` ; `chore:` ; `perf:` ; `build:` ; `ci:` ; `style:`.

## Exemples
- `feat(place): add themed search with clustering`
- `fix(photo): enforce 5MB limit and reject SVG`
- `refactor(tag): extract TagRepository interface`

## Règle
- Un commit **petit** et **cohérent**.
- **Bloqué** si `composer qa` n’est pas au vert.
